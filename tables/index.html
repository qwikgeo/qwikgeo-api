    Tables Endpoints

    Method URL Description
    GET https://api.qwikgeo.com/api/v1/tables Tables
    GET https://api.qwikgeo.com/api/v1/tables/{table_id} Table
    POST https://api.qwikgeo.com/api/v1/tables/ Create Table
    DELETE https://api.qwikgeo.com/api/v1/tables/{table_id} Delete Table
    POST https://api.qwikgeo.com/api/v1/tables/{table_id}/add_column Add Column
    DELETE https://api.qwikgeo.com/api/v1/tables/{table_id}/delete_column Delete Column

    Add Column

    Description

    The add column endpoints allows you to add a new column to an existing table in the database.

    Add Column endpoint is available at https://api.qwikgeo.com/api/v1/tables/{table_id}/add_column

    Example

    In the example below, we are adding a column called test that is text for the table vccvnkvhrmzsqqbbcacvjrlspfpdhbcthvjszbnfledgklxnps.

    Example Input

    {
    "column_name": "test",
    "column_type": "text"
}

    Example Output

    {
    "status": true
}

    Delete Column

    Description

    The delete column endpoint allows you to delete a column in an existing table in the database.

    Delete Column endpoint is available at https://api.qwikgeo.com/api/v1/tables/{table_id}/delete_column

    Example

    In the example below, we are deleting a column called test from the table vccvnkvhrmzsqqbbcacvjrlspfpdhbcthvjszbnfledgklxnps.

    Example Input

    {
    "column_name": "test"
}

    Example Output

    {
    "status": true
}

    Create Table

    Description

    The create table endpoints allow you to create a new table inside of a database.

    Create Table endpoint is available at https://api.qwikgeo.com/api/v1/tables/

    Example

    In the example below, we are creating a new table called vccvnkvhrmzsqqbbcacvjrlspfpdhbcthvjszbnfledgklxnps_new. We are adding one column in the table called postalcode, and setting the table to have POINT geometry.

    Example Input

    {
    "columns": [
        {
            "column_name": "postalcode",
            "column_type": "text"
        }
    ],
    "geometry_type": "POINT"
}

    Example Output

    {
    "status": true,
    "table_id": "vccvnkvhrmzsqqbbcacvjrlspfpdhbcthvjszbnfledgklxnps_new"
}
